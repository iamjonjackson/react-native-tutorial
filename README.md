# React Native + Supabase Workshop

## 🚀 Step 1 — Create your own copy

Click:

👉 Use this template  
👉 Create a new repository  
👉 Name it anything you like

---

## 💻 Step 2 — Open in Codespaces

Click:

Code → Codespaces → Create Codespace

Wait ~1–2 minutes for setup.

---

## ▶️ Step 3 — Start the app

In the terminal:

```
npx expo start
```

---

## 📱 Step 4 — Run on your phone

1. Install **Expo Go**
2. Scan the QR code
3. Your app should load instantly

---

## ✅ You’re ready

If you see:

✅ “Expo is running”

You’re good to go.

---
